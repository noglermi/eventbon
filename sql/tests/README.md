\# SQL Test Queries



Diese Dateien dienen zur manuellen Prüfung der Supabase-Daten während der Entwicklung.



\## Nutzung



1\. Supabase öffnen

2\. SQL Editor öffnen

3\. Dateiinhalt einfügen

4\. Run klicken



\## Dateien



\- 001\_sales\_overview.sql – letzte Verkäufe

\- 002\_payment\_summary.sql – Bar/Karte Zusammenfassung

\- 003\_sale\_items.sql – Verkaufspositionen

\- 004\_product\_revenue.sql – Umsatz je Produkt

\- 005\_revenue\_by\_hour.sql – Umsatz je Stunde



Diese Queries verändern keine Daten.
